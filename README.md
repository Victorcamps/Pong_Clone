# Pong Clone
A faithful recreation of the classic Pong arcade game built with C++ and SFML.

It includes a 1v1 and 1vbot function;

## How to run
1. Install SFML
2. Link SFML with your Visual Studio [in case you need help, click here](https://www.youtube.com/watch?v=lFzpkvrscs4&t=127s)
3. Clone the repo
4. Compile with your preferred C++ compiler
5. Run the executable
6. Have fun!
